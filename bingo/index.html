About Borderlands 2 Bingo

This is a bingo board for Borderlands 2.

This bingo run is designed around the Ungeared UVHM Zer0 speedrun, however it can be done using any save file or character desired.

Bingo starts after turning in "Bright Lights, Flying City" to Roland.

If using Ungeared UVHM characters, the character should start with a Vladof Rocket Launcher, a white Jakobs shotgun, a Lascaux, and Miss Moxxi's Bad Touch.

Save file for Ungeared UVHM Zer0 can be found here. This characters has all of the correct gear, and is set up at the proper place in the story.

Save file for Geared UVHM Sal can be found here. This characters has all of the correct gear, and is set up at the proper place in the story.

Use the load remover in LiveSplit to keep an accurate, loadless time.

About Video Game Bingo

To win a "Bingo" race, you must complete 5 of the tasks in a row horizontally, vertically, or diagonally.

The seed number is used to generate the board. Changing the seed will make a new board.

You can click on the squares to turn them green and red. This may help you organize your route planning.
